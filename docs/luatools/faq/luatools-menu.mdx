# Using the LuaTools Menu

The LuaTools menu shows up on Steam store pages after the plugin is installed. Here's what you can do with it.

---

## Adding a Game

1. Go to the game's **Steam store page**
2. On the right side, click **"Add via LuaTools"**
3. Click **"Restart Steam"**

Steam will restart and the game will be added to your library.

---

## Removing a Game

1. Go to the game's **Steam store page**
2. Click the **LuaTools icon** (small icon on the right side)
3. Click **"Remove via LuaTools"**
4. Click **"Restart Steam"**

---

## Applying a Fix

Some games need extra patches to work correctly (online fixes, DRM bypasses, etc.).

1. Go to the game's **Steam store page**
2. Click the **LuaTools icon**
3. Click **"Fixes Menu"**
4. Select the fix you want to apply
