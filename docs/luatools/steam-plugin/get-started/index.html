Install LuaTools

There are two ways to install LuaTools: the easy method (recommended) and the manual method for advanced users.

Easy Method (Recommended)​

Open PowerShell and run this command:

irm "https://luatools.vercel.app/install-plugin.ps1" | iex

That's it. The script will:

  1. Ask if you want to install SteamTools (say yes if you haven't already)
  2. Install the plugin loader
  3. Install the LuaTools plugin
Not sure what PowerShell is?

Press Win + S, type PowerShell, and open it. Then paste the command above and press Enter.

Manual Method​

Advanced users only

This method has no support. If you get stuck, use the Easy Method instead.

Step 1: Install SteamTools​

Download and install it from: https://steamtools.net/

Step 2: Install the plugin loader​

You'll need to find and install this yourself. Searching YouTube for a LuaTools installation guide is the easiest way.

Important

SteamTools and LuaTools are not affiliated. Do not ask SteamTools support about LuaTools. Doing so will get you banned from both communities.

Step 3: Install the LuaTools plugin​

  1. Download the latest plugin zip from GitHub Releases
  2. Open your Steam folder (usually C:\Program Files (x86)\Steam)
  3. Go into the plugins folder
  4. Create a new folder inside it (name it anything you want)
  5. Extract the zip into that new folder
Finding the plugins folder easily

Open Steam → click Steam in the top left → plugin loader → Plugins → Browse Local Files

Step 4: Enable the plugin​

Go to: Steam (top left) → plugin loader → Plugins → toggle LuaTools on → Save Changes

How to Use It​

Add a game:

Go to the game's Steam store page → click "Add via LuaTools" on the right side → click "Restart Steam"

Remove a game:

Go to the game's Steam store page → click the LuaTools icon → click "Remove via LuaTools" → click "Restart Steam"

Apply a fix:

Go to the game's Steam store page → click the LuaTools icon → click "Fixes Menu"
